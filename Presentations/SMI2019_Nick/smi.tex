\begin{frame}{Evaluation?}

After all this developmental effort\footnote<.->{First GitHub commit
  Aug. 27, 2014. Most recent Feb. 28, 2019.}:

\begin{itemize}
\item
  is the extra longitudinal processing worth it? I.e., is it better than
  just processing data through the cross-sectional pipeline?
\item
  And, if so, do we process in native space or SST space?
\end{itemize}

Although ANTs processing has certain advantages (just as FreeSurfer has
advantages),

\begin{itemize}
\item
  can we determine if the cortical thickness measures are somehow
  ``better'' than FreeSurfer's?
\item
  And can we do it in a more general way than has been done previously?
\end{itemize}

\end{frame}
